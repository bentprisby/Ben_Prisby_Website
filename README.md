## bentprisby.com

Next.js portfolio with MDX content, typed SEO helpers, and Tailwind UI.

### Commands

```bash
npm run dev
npm run build
npm run start
```

### Content

- Posts: `content/posts/*.mdx`
- Projects: `content/projects/*.mdx`

### Routes

- `/` Home (featured projects, latest posts)
- `/about`
- `/projects` and `/projects/[slug]`
- `/blog` and `/blog/[slug]`
- `/contact`
